Front-End와 Back-End를 아우르는 다양한 개발 경험을 바탕으로 막힘 없는 직무 수행이 가능합니다.
웹 퍼블리싱의 기본 직무는 그래픽 디자인과 기술 구현간의 격차를 해소하고, UI/UX 디자인 와이어프레임을 실제 코드로 변환하는 것 입니다. 흔히 디자인과 개발의 사이에 위치하기 때문에 이 사이의 간격을 줄이는 역할이 필요 합니다.

저는 웹 에이전시에서 국립국어원, 중소기업청 등의 공공기관 프로젝트 참여 경험을 살려 누구나 쉽게 사용할 수 있는 코레일 웹/앱 서비스를 만들겠습니다.

저는 온라인 교육회사에서 공공기관, 대기업, 중소기업 등 다양한 고객사의 프로젝트에 참여한 경험을 살려 지속 가능한 코레일 웹/앱 서비스를 만들겠습니다.

기존 레거시 웹 퍼블리싱 마크업 방식을 새롭게 바꾸겠습니다.
첫째, Node.js 기반의 템플릿 엔진인 EJS를 도입하여 여러 페이지에서 사용하는 부분을 컴포넌트화 하여 코드의 재사용성을 높이고 업무 생산성 및 유지보수 편의성을 대폭 높이겠습니다.
둘째, SASS를 도입하여 css의 구조적인 취약점을 해결하고 컴파일 과정을 통해 잘못된 코드의 발생을 사전에 막겠습니다.
셋째, Gulp를 도입하여 EJS의 html변환이나 SASS 컴파일, 이미지 최적화, js 파일 사이즈 최적화 등 반복적인 작업들을 자동화 하여 업무 생산성을 대폭 높이겠습니다.
넷째, NPM으로 Javascript 라이브러리들의 버전을 관리하여 협업자들간의 서로 다른 버전의 라이브러리 사용에 따른 오류 발생을 사전에 방지하겠습니다.

새로운 방식에 맞게 마크업 스타일 가이드를 만들어 동료들이 쉽게 적용할 수 있고, 일관된 방식과 코드로 만들어질 수 있도록 하겠습니다.

마지막으로, 개발자의 기본 덕목인 새로운 기술에 관심을 갖고 지속적인 발전을 이룰 수 있도록 하겠습니다.
